# Harness

Harness is a stack-agnostic bootstrapper that installs a spec-driven development workflow into any project. To set up spec-driven development in your project, run `bash bootstrap.sh` at the repository root. For more information on the design and philosophy behind Harness, see [docs/specs/2026-06-17-harness-design.md](docs/specs/2026-06-17-harness-design.md).
